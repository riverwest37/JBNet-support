# JB Net（爺婆Net）

Raspberry Pi / Pico W / Node-RED を活用した、  
家庭向け IoT 実践プロジェクトです。

本プロジェクトでは、テレビ表示・来客通知・監視画面・各種自動化など、  
家庭内で役立つシステムを低コストで構築します。

---

## 収録内容

### picow/

Raspberry Pi Pico W を使用した送信装置です。

- 押しボタン入力
- MQTT送信
- 来客通知イベント送信
- Wi-Fi接続動作

---

### nodered/

Node-RED フロー集です。

- TV電源ON/OFF
- HDMI切替
- MQTT受信
- 自動制御

---

### monitor/

テレビ表示・監視画面系です。

- 4画面表示
- カメラ監視
- 情報表示
- スケジュール表示

---

### docs/

補足資料・解説資料です。

- 配線図
- セットアップ手順
- 操作説明
- Q&A

---

## 特徴

- Raspberry Pi 活用
- Node-RED による簡単制御
